<h1 align="center">🍕 Invoice Automation in Excel</h1>

<p align="center">
  <b>An Interactive Restaurant Billing System built using Microsoft Excel</b><br>
  Automates invoice generation, tax calculation, item selection, and billing using Excel formulas and form controls.
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
This project demonstrates how Microsoft Excel can be transformed into a powerful
invoice automation tool without using complex software.
It is designed for restaurant billing where users can generate invoices simply by
selecting menu items and entering quantities.
</p>

<p>
The workbook automatically calculates item amounts, GST, subtotal, and final bill,
making the billing process faster, accurate, and user-friendly.
</p>

<h2>⚙️ Workflow</h2>

<ol>
<li>Enter customer information.</li>
<li>Select menu items using checkboxes.</li>
<li>Enter quantity for selected items.</li>
<li>Excel automatically calculates item amount.</li>
<li>Select GST type (Local/Central).</li>
<li>Tax is calculated automatically.</li>
<li>Grand Total is generated instantly.</li>
<li>Invoice is ready for printing or sharing.</li>
</ol>
